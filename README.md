# discordbot-js 

# Command Handler + Slash Discord Js v13

# Node.js 16.6.0 

# Installation ```npm install```



### You need to create config.json file 

- config.json
```js
{
    "token": "YOUR_BOT_TOKEN",
    "botID": "YOUR_BOT_ID",
    "prefix": "YOUR_PREFIX",
    
}
```
